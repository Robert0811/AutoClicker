# AutoClicker Pro

AutoClicker profesional pentru Windows cu interfață modernă.

## 📥 Instalare Rapidă

1. **Descarcă tot folderul**
2. **Dublu-click pe `SETUP.bat`**
3. Așteaptă instalarea automată
4. Gata! Vei avea shortcut pe Desktop

## 🎮 Utilizare

- **Dublu-click** pe "AutoClicker Pro" de pe Desktop
- SAU rulează `Porneste_AutoClicker.bat`
- Apasă **Z** pentru a porni/opri clicking-ul
- Reglează viteza cu butoanele **+** / **-** sau bara slider
- Viteză ajustabilă: **1-500 click-uri pe secundă**

## ✨ Funcții

- ⚡ Click automat până la 500 CPS
- 🎨 Interfață modernă dark theme
- 📊 Statistici în timp real (click-uri totale, timp activ)
- ⌨️ Shortcut global (tasta Z)
- 🔄 Reset statistici
- 🎯 Funcționează pe orice Windows (7/8/10/11)

## 📋 Cerințe

- Windows 7 sau mai nou
- Python se instalează automat dacă lipsește

## 🚀 Pentru prieteni

Trimite tot folderul și spune-le să ruleze **SETUP.bat** - se instalează singur!
